# AGENTS.md — CSV Equity Price Pipeline

## Overview
This repo stores daily US equity OHLCV price history in `price/*.csv`. The primary downloader is `download_yfinance.py`. It builds a wide ticker universe from NASDAQ Trader symbol directories and fetches `period="max", interval="1d"` from yfinance.

## Price data
- One CSV per ticker in `price/`.
- Columns: `Date,Open,High,Low,Close,Volume,Dividends,Stock Splits`.
- Rows use `pd.Timestamp` UTC `Date` index. Numeric columns are rounded to 5 decimals.
- Pre-existing non-empty `price/*.csv` are skipped on subsequent runs (resumable). Pass `--force` to re-download.

## Universe
`universe.csv` is generated by `download_yfinance.py` from:
1. `https://www.nasdaqtrader.com/dynamic/symdir/nasdaqtraded.txt`
2. `https://www.nasdaqtrader.com/dynamic/symdir/otherlisted.txt`
3. Optional: `--include-recent-delisted` adds symbols from `TradingSystemAddsDeletes.txt` DELETE actions.
4. Optional: `--alpha-vantage-key` augments with Alpha Vantage `LISTING_STATUS` equities.

Filters: exclude test issues, ETFs, units, warrants, preferreds. Deduplicated on `symbol`.

## Scripts
- `download_yfinance.py`: build universe + download/resume price history. Rate limit ~0.3s between tickers with 1 retry on failure.
- `alpha_utils.py`: `AlphaDataLoader` merges `price/*.csv` with `factCsv/*.csv` via `ticker_map.json`.
- `tools/data_loader.py`: simpler price-only loader.
- `run_alpha_mining.py`, `alphas.py`, `research_bottoms_v4.py`: alpha research/backtests.

## Tests
Run with:
```
python3 test_loader.py
python3 test_engine.py
python3 test_alphas.py
```
If these are missing, use `pytest` or `python3 -m unittest discover`.

## Commands
```bash
# Build universe and download missing tickers
python3 download_yfinance.py

# Re-download everything
python3 download_yfinance.py --force

# Skip universe rebuild
python3 download_yfinance.py --no-build-universe

# Include recently delisted symbols
python3 download_yfinance.py --include-recent-delisted

# Add Alpha Vantage symbols (requires key)
python3 download_yfinance.py --alpha-vantage-key YOUR_KEY
```

## Notes
- `universe.csv` is generated; do not hand-edit.
- `factCsv/` is populated by SEC EDGAR JSON extraction (`code-temp-old/batch_extract.py`) — separate from price download.
- Internet access is required for universe build and downloads.
- Expect ~25 minutes for a fresh run over ~6k tickers.
